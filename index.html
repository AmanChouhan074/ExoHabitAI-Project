About ExoHabitAI

Our Vision

What is ExoHabitAI?

ExoHabitAI is an advanced machine learning system designed to predict the habitability of exoplanets using astronomical data. It leverages modern AI algorithms and data-driven techniques to analyze planetary and stellar features — helping identify potentially habitable worlds beyond our solar system.

Project Goals

🎯 Objectives

Predict Habitability

Determine whether an exoplanet is habitable or not using supervised learning on real NASA exoplanet archive data.

Analyze Space Data

Process planetary radius, mass, orbital period, equilibrium temperature, stellar parameters for comprehensive scientific analysis.

Real-Time Predictions

Flask API delivers instant results — no delay, no friction. Input planet data via form or batch CSV and receive results immediately.

Scientific Exploration

Assist scientists, researchers and students in understanding the intersection of space science and artificial intelligence.
